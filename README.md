# Bank Customer Churn Prediction (ANN)

## 📌 Project Overview
This project predicts whether a bank customer will churn using an Artificial Neural Network (ANN).
The model is deployed using Streamlit and containerized using Docker.

## 🚀 Tech Stack
- Python
- TensorFlow / Keras
- Scikit-learn
- Streamlit
- Docker

## 🧠 ML Pipeline
- Data preprocessing (Label Encoding, One-Hot Encoding, Scaling)
- ANN model training
- Model evaluation
- Streamlit deployment
- Docker containerization

## ▶️ Run Locally
```bash
pip install -r requirements.txt
streamlit run app.py
